# Legacy Gauntlet Files

The files in `legacy/colab-originals/` are the original Gauntlet and
Motion-TimeSpace prototype materials that used notebook-style interfaces,
Colab widgets, and optional AI-provider wiring.

They were moved out of the repository root so new users land on the simple
non-AI app first. The files are intentionally preserved for reference and for
people who still want to run the earlier Colab workflows.

One file had a Windows-invalid `*` character in its GitHub filename. It has
been preserved under this safe name:

`This module contains an updated version of the Universal Paradox Resolution Engine.md`
